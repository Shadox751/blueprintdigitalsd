Print specs (when you pick one)

  • Final size: 3.5" × 2" (US standard)
  • Bleed: add 0.125" each side → 3.75" × 2.25"
  • Safe area: keep text 0.125" inside trim
  • Resolution: 300 DPI for any photo / raster
  • Color profile: CMYK (orange may shift slightly)
  • Stock: 16pt matte or 32pt soft-touch for premium feel
  • Print quantity: order 250 each (Allan + Miguel) to start
  • Vertical (#05) and tactile (#19, #12): consider spot-UV for accents
